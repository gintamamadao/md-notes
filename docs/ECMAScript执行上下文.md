# Detail

- 指当前执行环境中的变量、函数声明，参数（arguments），作用域链，this 等信息。
- 分为全局执行上下文、函数执行上下文，其区别在于全局执行上下文只有一个
- 函数执行上下文在每次调用函数时候会创建一个新的函数执行上下文。
