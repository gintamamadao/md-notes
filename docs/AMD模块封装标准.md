# Detail

- AMD 是"Asynchronous Module Definition"的简写，也就是异步模块定义。
- 基于 commonJS 规范的 nodeJS 出来以后，服务端的模块概念已经形成，很自然地，大家就想要客户端模块
- CommonJS 规范不适用于浏览器环境，因为它是同步的
- 浏览器端的模块，不能采用"同步加载"（synchronous），只能采用"异步加载"（asynchronous）。这就是 AMD 规范诞生的背景
- RequireJS 就是实现了 AMD 规范的
