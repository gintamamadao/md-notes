# null undefined 比较

- null == undefined 比较结果是 true
- null、undefined 和自己的比较结果为 true
- 除此之外，null、undefined 和其他任何结果的比较值都为 false

```js
null == undefined; // true
null == false; // false
null == undefined; // false
```
