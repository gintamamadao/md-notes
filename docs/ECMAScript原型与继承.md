# ECMAScript 原型与继承

- 每个函数都有一个属性 `prototype` 叫做原型，这个属性指向一个对象
- 调用构造函数所创建的实例对象会有一个 `__proto__` 属性指向构造函数的 `prototype`
- 构造函数的 `prototype` 就是实例的 原型
