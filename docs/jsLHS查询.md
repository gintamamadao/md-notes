# js LHS 查询

- 试图找到变量容器本身，为赋值操作找到目标
- 非严格模式下，如果在顶层作用域中无法找到目标变量，那么，就会在全局作用域中创建一个具有该名称的变量，并将其返回给引擎
  - 这种方式创建的全局变量，严格上来讲并不是真正的变量，而是全局对象的属性，可以通过 delete 操作符将其删除。
  - 但是，用 var 声明的全局变量，是不可以用 delete 操作符删除的
- 严格模式下，抛出 ReferenceError 异常
