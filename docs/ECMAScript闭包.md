# ECMAScript 闭包

- JavaScript 中所有的函数都是一个闭包，
- 有权访问上层作用域中的变量，
- 引用的外部变量不被释放，
- 但自己声明的私有变量不受外界干扰
- 但我们一般是指在函数中定义的函数，并且这个函数引用了父函数作用域的变量，并且在父函数之外被调用
- 引用的外部变量就会保留在词法作用域中
