# 当原始类型和引用类型做比较

- 当原始类型和引用类型做比较时，对象类型会依照 ToPrimitive 规则转换为原始类型

```js
"[object Object]" == {}; // true
"1,2,3" == [1, 2, 3]; // true
```

- 根据数组的 ToPrimitive 规则，数组元素为 null 或 undefined 时，该元素被当做空字符串处理，所以\[null]、\[undefined]都会被转换为 0

```js
[null] == false; // true
[undefined] == false; // true
```
